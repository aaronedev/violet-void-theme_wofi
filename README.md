# Violet Void Theme for Wofi

A dark purple-tinted theme for [Wofi](https://hg.sr.ht/~scoopta/wofi) launcher.

## Install

```bash
mkdir -p ~/.config/wofi/themes
cp violet-void-theme.css ~/.config/wofi/themes/

# In your config:
# style=~/.config/wofi/themes/violet-void-theme.css
```
